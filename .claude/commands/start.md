# スマートワークセッション

プロジェクトの状態を総合的に分析して、最適な作業フローを自動実行します。

## 分析項目

1. **Git状態分析**
   - 現在のブランチ名とパターン
   - 未コミット変更の有無
   - 最新コミットメッセージ

2. **Issue状態確認**
   - ブランチ名からissue番号抽出
   - Issue詳細の取得と分析
   - 実装要件の理解

3. **プロジェクト状態**
   - 依存関係の状態
   - ビルド・テスト状態
   - 型定義の最新性

## 実行ワークフロー

### フィーチャーブランチでの作業
```bash
# 自動実行されるコマンド例
git status
npm run fetch-issue [自動検出されたissue番号]
npm run generate-types  # 型定義更新
npm run ci:lint && npm run ci:type-check  # 事前品質チェック
```

### 実装フェーズ
- TodoWriteツールでタスク分解
- 段階的実装（実装→テスト→品質チェック）
- リアルタイムでの進捗管理

### 完了フェーズ
- 全品質チェック実行
- PR作成準備
- Root Cause Analysis記載

## Refeel開発ルール適用

- Vue 3 + TypeScript厳格モード
- Supabase認証セキュリティ確保
- 最小単位での段階的実装
- 新機能は必ずユニットテスト作成

現在の状況を総合的に分析して、最適な開発ワークフローを開始してください。
