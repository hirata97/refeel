# 自動ワークフロー実行

現在のgitブランチやプロジェクト状態を分析して、適切な作業を自動的に開始します。

## 実行フロー

1. **ブランチ分析**: 現在のブランチ名からissue番号を抽出
2. **Issue取得**: `npm run fetch-issue` でissue詳細を取得
3. **要件分析**: issue内容を分析して実装計画を作成
4. **段階的実装**: TodoWriteツールでタスク管理しながら実装
5. **品質チェック**: `npm run ci:all` で全品質チェック実行
6. **PR作成**: 実装完了時に `npm run create-pr` でPR作成

## 自動判定ロジック

- `feature/issue-XXX-*` ブランチ → issue XXX番の作業を開始
- `main` ブランチ → 新しいフィーチャーブランチ作成を提案
- 未コミット変更がある場合 → 継続かstashかを確認

## CLAUDE.mdの開発ルールに従って実装

- TypeScript厳格モード遵守
- Vue 3 Composition API使用
- 段階的実装（最小単位での確認）
- テスト同時作成
- 既存コードパターンの踏襲

現在のコンテキストを分析して、最適な作業を自動的に開始してください。
