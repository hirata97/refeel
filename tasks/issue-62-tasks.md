# Issue #62: 認証フローのE2Eテスト実装

## 概要
## 概要
Playwrightを使用して認証関連のE2Eテストを実装します。

## 実装内容
### 🔐 認証フローのテスト
- ログインフロー
  - 正常系：正しい認証情報でのログイン
  - 異常系：誤った認証情報でのログイン
  - バリデーションエラー
- ログアウトフロー
  - ログアウト処理
  - リダイレクト確認
- アカウント登録フロー
  - 新規ユーザー登録
  - 重複メールアドレスチェック
  - 必須項目バリデーション

### 📝 実装ファイル
- `e2e/auth.spec.ts`（新規作成）
- `e2e/helpers/auth.ts`（認証ヘルパー関数）
