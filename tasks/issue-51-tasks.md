# Issue #51: リマインダーと通知機能の実装

## 概要
## 概要
日記の継続的な記録を促進するため、リマインダーと通知機能を実装します。

## 現状の問題
- ❌ 日記記入の忘れを防止できない
- ❌ 定期的な振り返りの促進ができない
- ❌ 目標達成の通知ができない
- ❌ カスタマイズ可能な通知設定がない

## 実装内容

### ⏰ **リマインダー基本機能**
- 日記記入リマインダー
- カスタム時間設定
- 繰り返し設定（毎日、平日のみ等）
- スヌーズ機能

### 🔔 **通知システム**
- ブラウザプッシュ通知
- PWA通知（オフライン対応）
- 通知履歴管理
- 通知音カスタマイズ

### ⚙️ **設定機能**
- 通知時間のカスタマイズ
- 通知頻度の設定
- 通知方法の選択
- 一時的な通知停止

### 📊 **進捗通知**
- 連続記録達成通知
- 目標達成通知
- 週間/月間サマリー通知
- モチベーション強化メッセージ

## 技術要件
- Web Push API
- Service Worker
- PWA対応
- LocalStorage/IndexedDB

## 実装アプローチ
1. 基本的な通知システムの構築
2. リマインダー設定UIの実装
3. PWA対応の実装
4. 進捗通知システムの実装
5. 設定画面の拡張
6. パフォーマンス最適化

## 修正が必要なファイル
- `src/views/SettingPage.vue`
- `src/service-worker.js`（新規作成）
- `src/components/NotificationSettings.vue`（新規作成）
- `src/utils/notifications.ts`（新規作成）
